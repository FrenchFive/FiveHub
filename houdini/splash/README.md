# Splash

`fivehub_splash.png` is generated **on this machine** and stays out of
git: the installer renders it, and so do `python -m fivehub.cli splash`
and **FIVE HUB ▸ Pipeline Tools ▸ Regenerate FiveHub Splash** inside
Houdini. It renders with the Satoshi fonts downloaded into
`assets/fonts/`, so the pixels differ from machine to machine —
committing it would make every clone permanently dirty.
